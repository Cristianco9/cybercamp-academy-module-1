## Program Information

**Organization:** PrimeTek Africa - Cybercamp Academy

**Program:** Cybersecurity Professional Program  

**Modules:**

• **Module 1:** Cybersecurity Foundations  
(Networking, Linux, Security Principles)

• **Module 2:** Ethical Hacking & Penetration Testing  
(Kali Linux, Metasploit, Burp Suite)

• **Module 3:** Defensive Security & SOC Operations  
(SIEM, Incident Response, Threat Monitoring)

• **Module 4:** Cloud & Application Security  
(AWS Security, Azure Security, OWASP Top 10)

• **Module 5:** Certification & Career Preparation  
(CompTIA Security+ Preparation)

**Instructor:** Cristian Camilo Cortes Ortiz

**Instructor Role:** Systems Engineer & Cybersecurity / Software Development Instructor

This repository contains the official learning materials used during the 
**Cybersecurity Professional Program at Cybercamp Academy**.

It serves as a centralized location where students can access the labs, 
practice exercises, scripts, demonstrations, and security challenges used 
throughout the training modules.

## About Codecamp Academy

**Codecamp Academy** is focused on providing practical and hands-on technical 
education designed to prepare students for careers in the technology and 
cybersecurity industries.

The programs emphasize:

- Practical cybersecurity skills
- Understanding how networks, systems, and applications work
- Ethical hacking methodologies
- Defensive security strategies
- Incident response and threat analysis
- Secure development practices
- Collaborative work using Git and GitHub

Throughout the program, students work with **real-world security scenarios**, 
labs, and practical exercises that help them develop the skills required to 
work in modern cybersecurity environments.
